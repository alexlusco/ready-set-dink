Ready, Set, Dink!

1. Get the class landing URL by clicking "more info" button on the class you want to register for.

2. Paste URL into step 1.

3. If it worked, generated URL should say say "Oops... sorry, an error has occurred".

4. On registration day, log in to account, and navigate to the generated URL (fastest using Chrome browser). When registration opens, refresh the page by hitting command + R (mac) or ctrl + R (pc).

5. Finish registration, quick!

6. Pickle.
